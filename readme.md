# DevForum AI

A full-stack developer platform combining the best of Reddit, HackerRank, and StackOverflow into one ecosystem for collaborative learning and problem solving.

---

## Features

* Reddit-style discussions with vote-based comment ranking
* Collaborative problem solving through discussions and hints
* Problem set system with difficulty levels and test cases
* Community system for topic-based interaction
* Reputation and gamification system with badges and leaderboard
* Secure authentication using JWT
* Modern dark-themed responsive UI

---

## Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Flask (Python)
* Database: MySQL (XAMPP) (planned migration to PostgreSQL)
* Authentication: JWT

## Installation and Setup

### 1. Clone the Repository

```
git clone https://github.com/your-username/devforum-ai.git
cd devforum-ai
```

### 2. Create Virtual Environment

```
python -m venv venv
venv\Scripts\activate
```

### 3. Install Dependencies

```
pip install -r requirements.txt
```

### 4. Setup Database (XAMPP MySQL)

* Create a database named `devforum`
* Import the provided SQL file

### 5. Run the Application

```
python app.py
```
